A lightweight, native Android screen recording application built with Kotlin. It leverages Android's MediaProjection API and provides a beautiful, non-intrusive floating pill overlay inspired by Samsung's One UI design language.

Features:
• Native Performance — Built entirely in Kotlin for maximum efficiency
• One UI Aesthetic — Frosted-glass, draggable floating control pill
• Pause & Resume — Pause and continue recordings from the floating pill
• Dual Audio Capture — Record mic, system audio, or both simultaneously with real-time PCM mixing
• Studio-Grade Audio — 48kHz / 320kbps AAC encoding
• Quick Settings Tile — Start and stop recordings from your status bar
• Visual Countdown — Beautiful 3-second on-screen countdown before recording
• Material You Icon — Adaptive icon with monochrome support for dynamic theming
• Video Categorization — Tab-based filtering (All / Recents)
• Scoped Storage — MediaStore-based saving for broad device compatibility
• Automatic Organization — Saves MP4 recordings to DCIM/ScreenRecorder
